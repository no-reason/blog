### D. 花之少年
- **每个测试的时间限制**：2 秒
- **每个测试的内存限制**：256 兆字节

花之少年有一个由 `n` 朵花组成的花园，这个花园可以用一个整数序列 `a1, a2, …, an` 来表示，其中 `ai` 表示从左边数第 `i` 朵花的美丽值。

伊戈尔想要恰好采集 `m` 朵花。为此，他将从左到右走过花园，并决定是否采集他当前位置的花。他采集的花中，第 `i` 朵花的美丽值必须至少为 `bi`。

伊戈尔注意到，有可能无法采集到满足他美丽值要求的 `m` 朵花，所以在他开始采集花之前，他可以选择任意一个整数 `k`，并用他的魔法棒种出一朵美丽值为 `k` 的新花，并将它放置在花园中的任意位置（两朵花之间、第一朵花之前或最后一朵花之后）。由于他的魔法能力有限，他最多只能进行一次这样的操作。

请输出伊戈尔在执行上述操作时必须选择的最小整数 `k`，以确保他能够采集到 `m` 朵花。如果他无需进行该操作就能采集到 `m` 朵花，则输出 `0`。如果即使进行了该操作也无法采集到 `m` 朵花，则输出 `-1`。

### 输入
- 输入的第一行包含一个整数 `t`（`1 ≤ t ≤ 10^4`）—— 测试用例的数量。
- 每个测试用例的第一行恰好包含两个整数 `n` 和 `m`（`1 ≤ m ≤ n ≤ 2×10^5`）—— 花园中花的数量以及伊戈尔想要采集的花的数量，分别对应。
- 每个测试用例的第二行包含 `n` 个整数 `a1, a2, …, an`（`1 ≤ ai ≤ 10^9`）—— 其中 `ai` 是花园中从左边数第 `i` 朵花的美丽值。
- 每个测试用例的第三行包含 `m` 个整数 `b1, b2, …, bm`（`1 ≤ bi ≤ 10^9`）—— 其中 `bi` 是伊戈尔将采集的第 `i` 朵花必须具有的最小美丽值。

保证所有测试用例中 `n` 的总和不超过 `2×10^5`。

### 输出
对于每个测试用例，在单独的一行上输出伊戈尔在执行上述操作时必须选择的最小整数 `k`，以确保他能够采集到 `m` 朵花。如果他无需进行该操作就能采集到 `m` 朵花，则输出 `0`。如果即使进行了该操作也无法采集到 `m` 朵花，则输出 `-1`。

### 说明
- 在第一个测试用例中，假设伊戈尔种出一朵美丽值为 `6` 的花，并将它放置在第三朵花和第四朵花之间。那么，花园将如下所示：`[3,5,2,6,3,3,5,8,1,2]`。然后，他可以选择第二朵、第四朵、第六朵、第七朵和第八朵花，它们的美丽值分别为 `[5,6,3,5,8]`。
- 在第三个测试用例中，他可以种出一朵美丽值为 `7` 的花，并将它放置在第一朵花之前。花园将如下所示：`[7,4,3,5,4,3]`。现在，他可以选择第一朵、第二朵和第四朵花。
- 在第四个测试用例中，伊戈尔无需进行该操作，所以答案是 `0`。
- 在第六个测试用例中，无论伊戈尔如何进行该操作，他都无法采集到 `3` 朵花，使得他采集的第 `i` 朵花的美丽值至少为 `bi`，因此答案是 `-1`。 


分析:
本题的关键在于考虑前k个a_i能至多匹配到的b_j的下标和后k个a_i能之多匹配到的b_j的下标这两个数组
``` c++
#include<iostream>
#include<cstdio>
#include<algorithm>
#include<string>
#include<cstring>

using namespace std;

int T;
int n,m;
const int N=2e5+13;
int a[N];
int b[N];
int front[N];
int back[N]; 
int main(){
	cin>>T;
	while (T--){
		int ans=0x3f3f3f3f;
		bool flag=false;
		scanf("%d%d",&n,&m);
		for (int i=1;i<=n;i++){
			cin>>a[i];
		}
		for (int i=1;i<=m;i++){
			cin>>b[i];
		}
		int now=1;
		for (int i=1;i<=n;i++){
			if (a[i]>=b[now]){
				front[i]=now;
				now++;
			}
			else
				front[i]=front[i-1];
		}
		if (front[n]>=m){
			printf("%d\n",0);
			continue;
		}
		int ns=m;
		for (int i=1;i<=n+1;i++){
			back[i]=m+1;
		} 
		for (int i=n;i>=1;i--){
			if (a[i]>=b[ns]){
				back[i]=ns;
				ns--; 
			}
			else {
				back[i]=back[i+1];
			}
		}
		for (int i=0;i<=n;i++){
			if (front[i]+2==back[i+1]){
				ans=min(ans,b[front[i]+1]);
				flag=true;
			}
		}
		if (flag)
			cout<<ans<<endl;
		else 
			cout<<-1<<endl;
	} 
	return 0;
}
```
